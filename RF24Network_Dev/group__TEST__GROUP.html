#define MIN_USER_DEFINED_HEADER_TYPE   0

Network Management message types for management of network frames and messages System discard types (128 to 147) Contain no user data, just additional system sub-types sent for informational purposes.

(Initially within NETWORK_ACK responses) System retained types (148-167) Contain user data

System types can also contain sub-types, included as information, TBD
