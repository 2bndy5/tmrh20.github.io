void sleep_c::begin ( wdt_prescalar_e  prescalar,
short  cycles 
)

Enable the system to be able to sleep.

Does not actually do any sleeping.

For example, to do something roughly evert minute, configure it like this:

Sleep.begin(wdt_8s,7);
Parameters
prescalarDuration of the watchdog timer interrupt. The system will actually sleep for this long.
cyclesNumber of times the system will wake up before returning from go().
Examples:
sensornet.pde.
