201  * RF24Gateway is a library style replacement for the standalone RF24toTUN application, used with RF24Ethernet.
202  *
203  * In addition to passing TCP/IP payloads to a network interface, RF24Gateway also allows users to send and receive
204  * standard RF24Network payloads.
205  *
206  * This allows users to deploy and manage hybrid networks, consisting of nodes communicating via TCP/IP and RF24Network Messages
207  *
208  * @section Installation Installation
209  *
210  * Installation instructions will be included with the RF24Ethernet documentation at http://tmrh20.github.io/RF24Ethernet once RF24Gateway
211  * is completed.
212  *
213  * **Current:**
214  *
215  * a: Install RF24 libs per installer. <br>
216  * b: Manually download/clone RF24Gateway from https://github.com/TMRh20/RF24Gateway and run 'sudo make install' <br>
217  * c: Edit one of the examples to specify your CE/CSN pin config ( RF24 radio(22,0); ) <br>
218  * d: If using the RF24GatewayNode example, edit to set your IP/Subnet information <br>
219  * e: 'make' and run the example file(s) <br>
220  *
221  * Note: Forwarding/NAT may need to be configured as with RF24toTUN<br>
