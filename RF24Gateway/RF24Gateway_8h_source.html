136 /**
137  * @mainpage RF24Gateway
138  *
139  * RF24Gateway is a library style replacement for the standalone RF24toTUN application, used with RF24Ethernet.
140  *
141  * In addition to passing TCP/IP payloads to a network interface, RF24Gateway also allows users to send and receive
142  * standard RF24Network payloads.
143  *
144  * This allows users to deploy and manage hybrid networks, consisting of nodes communicating via TCP/IP and RF24Network Messages
145  *
146  * @section Installation Installation
147  *
148  * Installation instructions will be included with the RF24Ethernet documentation at http://tmrh20.github.io/RF24Ethernet once RF24Gateway
149  * is completed.
150  *
151  * **Current:**
152  *
153  * a: Install RF24 libs per installer. <br>
154  * b: Manually download/clone RF24Gateway from https://github.com/TMRh20/RF24Gateway and run 'make install' <br>
155  * c: 'make' and run the example file <br>
156  * d: Edit the InterfaceConfig.sh file to configure your IP/Subnet and run it to set up the interface. <br>
157  *
158  * Note: Forwarding/NAT may need to be configured as with RF24toTUN<br>
159  * Note: RF24Gateway defaults to using RF24Mesh. Sensor nodes must also use RF24Mesh with the default configuration.
160  *
161  *
