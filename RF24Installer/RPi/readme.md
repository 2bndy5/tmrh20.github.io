
##RF24toTUN installer

###Installs the following libraries:
RF24
RF24Network
RF24Mesh
RF24toTUN

###Instructions
Download the install.sh file from http://tmrh20.github.io/RF24Installer/RPi/install.sh
1. ```wget http://tmrh20.github.io/RF24Installer/RPi/install.sh```

Make it executable:
2. ```chmod +x install.sh```

Run it:
3. ```./install.sh```
